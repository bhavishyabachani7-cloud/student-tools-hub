🚀 Student Tools Hub

Student Tools Hub is a modern, all-in-one web application built using Flask (Python 3.11) that provides a collection of useful tools for students, developers, and everyday users. It is designed with a clean UI, fast performance, and mobile responsiveness.
✨ Features
🧠 Core Tools
✅ BMI Calculator
✅ Discount Calculator
✅ Word Counter
🖼️ Image Tools
✅ Image Resizer
✅ Image Compressor
✅ Image to PDF Converter
📄 PDF Tools
✅ PDF to Text Converter
⚙️ Utility Tools
✅ Notes Formatter
✅ AI Bio Generator
✅ Resume Builder (PDF Download)

📱 Fully mobile responsive
